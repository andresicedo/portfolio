SortingVisualizer

This project is a visualization of sorting algorithms used in programming. The sorting algorithms displayed are bubble sort, insertion sort, quick sort and selection sort. An array of random numbers are visualized as a the cyan highlighted bars on the screen and the user can choose which sorting algorithm to sort the bars in ascending order. The user can choose how big or small they want their array input to be, as well as choose how fast they want the algorithm to execute.

This application was built using HTML, custom CSS, and Javascript

DevExpo

This is a social media application I built for developers and creators to display their work. Users can post their projects, show the community the technologies used to build the project, and provide a link to their project for users to check out. The app also allows users to comment on posts and leave feedback and questions about the project.

This project was built using React on the front end, Material UI as the CSS framework and Firebase for user authentication and back end functions.

PlatePool

Similar to Facebook Marketplace, this application allows users to list food that they are cooking, sell to other users and share their passion for cooking within their community.

This application was built primarily with React, Node.js, and Material UI, as well as SQL through Postgres and Sequelize for the database, AWS cloud technology to store images that users might upload and a Mapquest API to get the time/distance between users.
